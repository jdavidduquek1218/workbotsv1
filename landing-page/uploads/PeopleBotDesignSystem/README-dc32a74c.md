# 🎨 PeopleBot Design System

Sistema de diseño completo extraído de la landing page de PeopleBot.

## 📂 Contenido

### 📚 Documentación
- **DESIGN_SYSTEM.md** - Especificación completa del sistema (colores, tipografía, componentes, animaciones)
- **IMPLEMENTATION_GUIDE.md** - Guía práctica de implementación y uso
- **README.md** - Este archivo

### 🎯 Archivos Principales
- **design-system.json** - Estructura JSON completa del sistema de diseño (importable)
- **design-system-interactive.html** - Guía visual e interactiva (abre en navegador)

### 🎨 Estilos CSS
- **css/design-tokens.css** - Variables CSS reutilizables
- **css/components.css** - Componentes listos para usar
- **css/utilities.css** - Clases utilitarias

## 🚀 Quick Start

### 1. Ver la guía visual
Abre `design-system-interactive.html` en tu navegador para ver todos los componentes, colores y animaciones.

### 2. Usar en tu proyecto web
```html
<link rel="stylesheet" href="css/design-tokens.css">
<link rel="stylesheet" href="css/components.css">
<link rel="stylesheet" href="css/utilities.css">
```

### 3. Importar en herramientas de diseño
Usa `design-system.json` para importar en tu herramienta de diseño preferida.

## 📋 Qué Incluye

✅ Paleta de colores (primarios, acentos, neutros)
✅ Tipografía (Space Grotesk + Plus Jakarta Sans)
✅ Sistema de espaciado (11 niveles)
✅ Componentes (botones, cards, badges, grids)
✅ Animaciones (fadeUp, floatCard, blink)
✅ Efectos hover y transiciones
✅ Grid system (2, 3, 4 columnas)
✅ Responsive design (768px breakpoint)
✅ Sombras y efectos visuales
